# Fonts

## JF Flat Jozoor (Arabic)

Place the JF Flat Jozoor font file here:

- `JFFlatJozoor.ttf`   ← **required**

The font is available from:
  https://ar.arabicfonts.net/fonts/jf-flat-jozoor

Once placed, the browser will automatically use JF Flat Jozoor for all Arabic text
(`dir="rtl"` or `lang="ar"` elements) and for English headings on the login page.

Until the file is present, the site falls back to Cairo (Google Fonts).
