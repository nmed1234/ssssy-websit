Design Philosophy — Official Website Standard

All motion follows "purposeful restraint" — the standard of UN, WHO, World Bank, national universities. Animations serve information, never decorate.

✅ Permitted
  • Fade-in on scroll (400ms, once)
  • Translate-up on scroll (20px → 0)
  • Card stagger entrance (50ms delay)
  • Hover: lift 3px + shadow deepen (200ms)
  • Input focus: border + shadow (150ms)
  • Thin scroll progress bar
  • Animated counters (once on entry)
  • Hero heading word-reveal (once)
  • PDF page-flip (user-triggered)
  • Timeline SVG line draw (scroll-tied)
❌ Forbidden in ALL Themes
  • Looping floating blobs / shapes
  • Hero parallax background
  • Spring bounce / elastic physics
  • Magnetic cursor effects
  • Gradient shift background loop
  • Particle fields on content
  • Pulsing glow on rest state
  • Shimmer loops on elements
  • Scale breathe / continuous loops

Note: ParticleField is kept only in Classic Soil (existing behavior). Hidden by CSS for all other themes.

Advanced Smooth Effects Applied in All Themes

1
Frosted Surface — backdrop-filter: blur(0–20px). Degree varies: flat themes use 0px, glass themes use 8–16px. Creates depth without noise.
2
2-Stop Layered Shadows — Every card uses ambient + direct shadow (two box-shadow values). Hover deepens both. Creates realistic lift.
3
Standard Easing Curve — cubic-bezier(0.4, 0, 0.2, 1) on all transitions. Clean deceleration. No bounce, no elastic, no spring.
4
Soft Border Opacity — Borders use rgba at 15–40% opacity. Cards sit softly on background rather than looking cut out.
5
Scroll-Linked Section Reveals — Framer Motion whileInView, once:true, margin:"-80px". Every section fades + rises on first entry only.
6
Reading Typography Tuning — line-height: 1.65–1.8, letter-spacing per theme, text-wrap:balance on headings. Each theme feels typographically complete.
7
3-Level Surface Hierarchy — Page background → card surface → elevated element. Each step is 4–8% lighter/darker for depth without using shadows alone.
8
Accessible Focus Ring — 3px solid primary color ring on every interactive element. Matches each theme's primary color via --style-shadow-focus.
9
Theme Switch Transition — :root transition on background-color, color, border-color at 300ms. Switching themes feels smooth not jarring.
