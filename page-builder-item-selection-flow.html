Page Builder — Item Selection

Two-tier interaction: select a section, then drill into individual items to edit their properties

1

Click a Section

Click any section on the canvas. It gets a blue border. Property panel shows section-level Content / Config / Styling tabs.

→
2

Click "Select item"

A Select item button appears in the top-right toolbar when a section with items is selected. Click it to enter item-pick mode.

→
3

Click an Item on Canvas

The section shows a numbered overlay grid. Each zone corresponds to one item. Click any zone to select that item.

→
4

Edit Item Properties

Property panel switches to Item Mode: Content tab shows all item fields (title, text, URL, button…), Style tab shows visual properties.

Canvas Changes

  • Selected section gets blue border
  • "Select item" violet button appears in toolbar
  • Item-pick mode shows numbered overlay grid
  • Hover shows item label badge + highlight
  • Click zone → exits item-pick, opens item editor

Property Panel Changes

  • Section mode: same Content / Config / Styling tabs
  • Items list: click any row → full item editor opens
  • Item mode: violet header + item mode badge
  • Content tab: title, text, URL, button, icon fields
  • Style tab: bg color, text color, border, padding
  • ← / → navigation between items
  • "Back to section" returns to section mode

Supported Section Types

  • section, row, flexbox, grid, accordion, list
  • card-group, team, testimonial, timeline
  • gallery, carousel, slideshow, lightbox
  • stats, counter, faq, tabs, navbar, footer
  • pricing-table, progress, breadcrumb

How Updates Are Saved

  • Item edits write to section.data[fieldKey][index]
  • Live-saved on every field change (no Save button needed)
  • Changes immediately reflected in canvas preview
  • Full JSON mode available for bulk edits
